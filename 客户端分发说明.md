# 智能记账客户端 - 完整分发包

## 项目概述

这是一个完整的智能记账客户端分发包，包含桌面应用程序和所有必要的依赖文件。

## 文件结构

```
智能记账客户端分发包/
├── 智能记账客户端.exe          # 主程序可执行文件
├── start_client.py             # 启动器脚本
├── simple_desktop_client.py    # 桌面客户端核心
├── build_client.py             # 构建工具
├── finance_system.db           # 数据库文件
├── install.bat                 # 安装脚本
├── README.md                   # 项目说明
├── 使用说明.txt                 # 用户使用说明
├── 客户端分发说明.md            # 分发说明
└── templates/                  # 网页模板
    ├── dashboard.html
    ├── login.html
    ├── register.html
    └── reports.html
```

## 功能特性

- **用户管理**: 注册、登录、多用户支持
- **记账功能**: 收入支出记录、分类管理
- **数据统计**: 月度汇总、图表展示
- **本地存储**: SQLite数据库，数据安全
- **Web界面**: 基于Flask的现代化界面
- **桌面集成**: 完整的桌面应用程序

## 使用方法

### 方法一：直接运行可执行文件

1. 双击运行 `智能记账客户端.exe`
2. 自动打开浏览器访问 http://127.0.0.1:5000
3. 使用测试账号: `testuser` / `test123`

### 方法二：使用Python脚本

1. 确保安装Python 3.7+
2. 运行 `pip install flask`
3. 运行 `python start_client.py`

### 方法三：使用安装脚本

1. 双击运行 `install.bat`
2. 自动安装依赖并启动程序

## 构建说明

如需重新构建客户端，运行：

```bash
python build_client.py
```

## 技术栈

- **后端**: Flask + SQLite
- **前端**: HTML + CSS + JavaScript
- **构建工具**: PyInstaller
- **数据库**: SQLite3

## 系统要求

- Windows 7/10/11
- Python 3.7+ (仅脚本运行需要)
- 4GB RAM
- 100MB 磁盘空间

## 技术支持

如有问题，请检查：

1. 端口5000是否被占用
2. 防火墙是否阻止程序运行
3. 是否安装了必要的Python包
