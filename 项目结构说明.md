# 智能记账客户端 - 优化后的项目结构

## 📁 项目文件结构

```
Finance/
├── 📄 核心文件
│   ├── backend_api.py          # 后端API服务 (端口: 5001)
│   ├── desktop_client.py       # 桌面客户端主程序 (端口: 5000)
│   ├── start_system.py         # 开发环境启动脚本
│   ├── build_client.py         # 打包工具
│   └── 智能记账客户端.spec      # PyInstaller打包配置
│
├── 🌐 前端模板
│   └── templates/
│       ├── dashboard.html      # 主仪表板 (已优化)
│       ├── login.html          # 登录页面
│       ├── register.html       # 注册页面
│       └── reports.html        # 报告分析页面
│
├── 📦 分发文件
│   ├── install.bat             # 安装脚本
│   ├── 使用说明.txt            # 用户使用说明
│   └── dist/                   # 打包输出目录
│       └── 智能记账客户端.exe   # 可执行文件
│
├── 📊 数据文件
│   └── finance_system.db       # SQLite数据库 (自动生成)
│
└── 📚 文档
    ├── README.md               # 项目说明
    ├── 记账客户端使用说明.md    # 详细功能说明
    └── 项目结构说明.md         # 本文件
```

### 新增的文件

## 🚀 使用方式

### 开发环境
```bash
# 方式1: 使用启动脚本
python start_system.py

# 方式2: 分别启动
python backend_api.py  # 后端服务
python desktop_client.py  # 桌面客户端
```

### 生产环境
```bash
# 方式1: 直接运行可执行文件
双击运行 dist/智能记账客户端.exe

# 方式2: 使用安装脚本
运行 install.bat
```

### 打包应用
```bash
# 生成可执行文件
python build_client.py
```

- ✅ **每日记账** - 完整的记账表单和快速操作

## 🔧 技术架构

### 后端技术栈
- **框架**: Flask
- **数据库**: SQLite
- **API**: RESTful
- **安全**: SHA-256密码加密

### 前端技术栈  
- **框架**: 原生JavaScript
- **图表**: ECharts
- **样式**: 现代CSS (Flexbox/Grid)
- **交互**: AJAX + Web API

### 打包技术
- **工具**: PyInstaller
- **格式**: Windows可执行文件
- **依赖**: 自动包含所有依赖
